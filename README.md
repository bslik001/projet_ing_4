# Projet ING 4

Collection de mini-applications Python réalisées en 4ème année d'ingénierie.

## Contenu

| Dossier | Description |
|---|---|
| `best_app/` | Application principale |
| `dashboard/` | Tableau de bord de visualisation |
| `india_ka/` | Analyse de données indiennes |
| `prediction_medicale/` | Prédiction médicale (machine learning) |
| `stock_forecast_app/` | Prévision de cours boursiers |
| `bmi.py` | Calculateur d'IMC |

## Installation

```bash
git clone https://github.com/bslik001/projet_ing_4.git
cd projet_ing_4
pip install -r requirements.txt
```

## Utilisation

```bash
# Calculateur IMC
python bmi.py

# Lancer une application
cd stock_forecast_app
streamlit run app.py
```
